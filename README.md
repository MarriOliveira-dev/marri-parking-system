# Marri Parking System 

Sistema de estacionamento inteligente desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades
✅ Registro de veículos  
✅ Controle por tipo  
✅ Validação de horário  
✅ Cálculo automático  
✅ Relatório financeiro  
✅ Histórico de registros  

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Objetivo do projeto
Projeto criado para praticar lógica de programação, front-end e desenvolvimento de sistemas web.
